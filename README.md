# יומן מסחר פורקס

אפליקציית יומן מסחר מתקדמת בעברית עם דשבורד, ניתוחים, פילטרים, ייבוא/ייצוא והתקנה כאפליקציה (PWA).

## הפעלה
```powershell
cd "C:\Users\דניאל רבי\.codex\forex-journal"
python -m http.server 8080
```
ואז לפתוח:
`http://localhost:8080`

## תכונות מרכזיות
- יומן עסקאות מלא עם עריכה/מחיקה.
- דשבורד מדדים (אחוז הצלחה, פקטור רווח, R ממוצע, משיכה מרבית, רצפים).
- גרפים: עקומת הון + ביצועים לפי יום בשבוע.
- לוח חודשי רווח/הפסד.
- ניתוחים לפי צמד, סשן ויום.
- צ'קליסט לפני כניסה + תיעוד טעויות.
- תגיות, רגש, הערות וצילום מסך לעסקה.
- ייצוא JSON/CSV וייבוא JSON/CSV.
- תמיכה ב-PWA + עבודה אופליין בסיסית.
